01 · How it works

One data pipeline, two sides of the same .env

A Windows-only Python bridge pulls trades from the MT5 terminal, enriches them and writes to Google Sheets. The Next.js app reads from Sheets, renders the analytics, and proxies live positions back from the bridge.

01 · WINDOWS

MT5 Terminal

terminal64.exe

Your broker's MetaTrader 5 — the source of every trade, fill and account snapshot.

02 · PYTHON

bridge.py

port 8765

FastAPI bridge pulls history, computes pips / session / duration / R, writes to Sheets.

03 · CLOUD

Google Sheets

google-spreadsheet

Acts as the database. Bot-written columns stay locked; your journal fields are yours to edit.

04 · WEB

Next.js App

port 3000

App Router UI + API routes. Reads Sheets, draws charts, proxies live positions from the bridge.

02 · The workspace

Seven views, one trading cockpit

Each view answers a different question — how am I doing, when do I trade well, and why.

Dashboard

KPI cards, equity curve, win-rate donut, daily P&L bars and a mini calendar with monthly-goal progress.

Calendar

Daily P&L heatmap with high- and medium-impact economic news events overlaid on each day.

Trades

Sortable, filterable blotter with a TradeSheet detail panel and editable journal notes per trade.

Positions

Live open positions and pending orders streamed in real time straight from the bridge.

Analytics

R-distribution, hour heatmap, weekday × session matrix, breakdowns by symbol & setup — plus emotion tracking.

Sheets

Monthly and per-symbol breakdown table that drills all the way down to the daily trades behind it.

Settings

Credentials, sync interval, monthly goal target and appearance — all backed by a single config file.

05 · Zero-config install

Three executables

MeridianInstaller.exe

First-time setup — installs Node, Python, MT5, all packages and auto-generates secrets.

MeridianSetup.exe

Health check — shows exactly what's missing or misconfigured before you launch.

Meridian.exe

Daily launcher — boots Next.js and the Python bridge together with one double-click.
